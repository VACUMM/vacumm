1.6.1.2. vacumm_print_config.py – Print your VACUMM configuration (DEPRECATED)¶

Please use vacumm_config.py – Print and edit your configuration instead.

This script is deprecated. Please use “vacumm_config.py print” instead .. program:: vacumm_config.py

Usage:
vacumm_config.py print [-h] [--system] [--direc] [--config]
        [--packages] [--no-header]
Description:
Optional Arguments:
 
-h, --help¶

show this help message and exit

--system, -s¶

Print only system information

--direc, -d¶

Print only important VACUMM directories

--config, -c¶

Print only VACUMM config itself

--packages, -p¶

Print only version of important packages

--no-header¶

Do not add headers
See also:

vacumm_edit_config.py – Edit your user configuration file (DEPRECATED), print_config().
