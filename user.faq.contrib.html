1.4.5.4. How to contribute to the documentation?¶

The documentation is written in the reStructuredText language, and generated and compilated with the sphinx python utility.

Just follow examples of the existing documentation sources in doc/sphinx/source to make modifications or add new content, then commit your files. If you want to compile the documentation, please go to section Generation of the sphinx documentation.
