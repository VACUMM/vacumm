1.1. Introduction¶

“Validation, Analysis, Comparison - Utilities written in Python to validate and analyze Multi-Model outputs, and compare them to observations.”

—Abstract translated from the forge page of the project.

VACUMM provides generic and specialized tools for the validation ocean models, and more especially the MARS model from IFREMER. The heart of VACUMM is a library written mainly in the Python language, whose core can be used for the preprocessing et le postprocessing of oceanic and atmospheric data coming from models or observations. The library for instance also has specialized modules for managing outputs from models and making advanced diagnostics.
